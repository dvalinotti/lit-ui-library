# Lit UI Library

Example project for using Lit Web Components as a tool for building UI component libraries.

## Includes

- Lit framework
- Vite
- ESLint + Prettier config
- Storybook

## Development

1. `git clone https://github.com/dvalinotti/lit-ui-library.git`
2. `yarn`
3. `yarn dev` OR `yarn storybook`
